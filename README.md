# colorizr

## Features Overview
Colorizr adds **color** to your strings.
Supported colors
  * `red`
  * `green`
  * `yellow`
  * `blue`
  * `pink`
  * `light_blue`
  * `white`
  * `light_grey`
  * `black`

## Usage Instructions
```
require 'colorizr'

puts 'John'.red
puts 'Paul'.green
puts 'George'.blue
puts 'Ringo'.yellow
```
